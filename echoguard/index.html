Bem Vindo a EchoGuard

Um software de segurança que utiliza a voz do usuário como uma chave de autenticação multifatorial para acesso a dispositivos e dados sensíveis.

Suas principais funcionalidades incluem:

  • Autenticação por Voz
  • Análise de Voz em Tempo Real
  • Proteção de Dados Sensíveis
  • Alertas e Notificações
  • Modo Silencioso
  • Relatórios de Segurança


Tecnologias que usamos


Python:

Para desenvolvimento de algoritmos de reconhecimento de voz e machine learning, linguagem de programação de alto nível, interpretada e de propósito geral, amplamente usada devido à sua simplicidade e versatilidade.

TensorFlow/Keras:

Para o desenvolvimento e treinamento de modelos de reconhecimento de voz, bibliotecas de código aberto populares usadas para construir e treinar modelos de aprendizado de máquina e redes neurais.

Acesse Aqui
Google Cloud Speech-to-Text

Para conversão de fala em texto, se for necessário um serviço de reconhecimento de voz baseado em nuvem, são dois serviços de reconhecimento de fala que permitem converter áudio.

Acesse Aqui
PostgreSQL:

Para armazenar perfis de usuário e logs de acesso, sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto e altamente avançado. Ele é conhecido por sua robustez, flexibilidade e conformidade com padrões SQL

Acesse Aqui
OAuth 2.0:

Para autenticação segura e autorização, protocolo de autorização que permite que aplicativos de terceiros acessem recursos de um usuário hospedados em um servidor, de maneira segura sem a necessidade de expor as credenciais do usuário

Acesse Aqui
PyTorch:

Alternativa para o desenvolvimento de modelos de deep learning, biblioteca de código aberto desenvolvida pelo Facebook para aprendizado de máquina e deep learning. É amplamente utilizada para criar e treinar modelos de redes neurais.
